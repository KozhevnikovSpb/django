1. last - возвращает последний элемент списка

{{ my_objects|last }}
{{ my_list|last }}

2. join - объединяет список строк в одну строку

{{ my_list|join:'' }}

3. escape - экранирует HTML

{{ my_text|escape }}

4. length - возвращает кол-во элементов в списке

{{ my_objects|length }}
{{ my_list|length }}

5. lower - преобразует строку в нижний регистр

{{ my_text|lower }}

6. upper - преобразует строку в верхний регистр

{{ my_text|upper }}

7. title - все слова с прописных(заглавных) букв

{{ my_text|title }}

8. random - возвращает рандомный элемент из списка

{{ my_list|random }}

9. truncatechars - обрезает строку до n символов

{{ my_text|truncatechars:7 }}

10. time - форматирует время с заданным форматом

{{ now_time|time:"H:i"}}
